# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how Bitcoin market sentiment (Fear/Greed index) affects
trader behavior and performance on the Hyperliquid platform.

## Dataset
1. Hyperliquid historical trade data
2. Bitcoin Fear/Greed index dataset

## Methodology
• Data cleaning and preprocessing
• Dataset merging by date
• Exploratory data analysis
• Visualization of trading behavior
• Insight extraction

## Key Findings
1. Trading participation decreases during extreme fear.
2. Many trades represent position adjustments with zero realized PnL.
3. Trade sizes vary significantly among traders.

## Strategy Recommendations
• Reduce position sizes during Fear markets.
• Increase participation during Greed sentiment.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook
